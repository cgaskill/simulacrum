# Simulacrum

This project uses react and webpack on the front end and spring boot and kotlin on the backend.

## Demo
https://simulacrum.herokuapp.com

## Get Started
```sh
cd ./api
mvn spring-boot:run
```

```sh
cd ./app
npm install
npm start
```

Then open [http://localhost:3000/](http://localhost:3000/) to see your app.
